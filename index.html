But what is Client Side Programming?

Client-side means that the action takes place on the user’s (the client’s) computer:

  • - Interact with temporary storage;
  • - Make interactive web pages;
  • - Interact with local storage;
  • - Sending request for data to server;
  • - sending requests to the server;
  • - Send request to server;
  • - work as an interface between server and user;
  • - Providing remote access for client-server program.
